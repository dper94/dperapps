Términos de Uso — Quiet Mode

Última actualización: 05/02/2026

Lee estos términos cuidadosamente antes de usar Quiet Mode ("la Aplicación"). Al usar la Aplicación, aceptas estos términos.

1. Uso de la Aplicación

  • Quiet Mode te permite bloquear aplicaciones y sitios web que distraen y crear sesiones de enfoque.
  • Debes usar la aplicación de manera legal y responsable.

2. Suscripciones

  • Algunas funciones requieren una suscripción premium, que se gestiona a través de Google Play / App Store.
  • Todas las compras y renovaciones están sujetas a los términos de la plataforma correspondiente.
  • Las cancelaciones deben realizarse de acuerdo con las políticas de Google Play / App Store.

3. Servicios de accesibilidad, notificaciones y análisis

  • La aplicación utiliza el servicio de accesibilidad para detectar aplicaciones activas y habilitar sesiones de enfoque.
  • La aplicación utiliza el servicio de notificaciones para identificar notificaciones que distraen y bloquearlas.
  • La aplicación utiliza Firebase Analytics para recopilar datos de uso anonimizados (interacciones con la aplicación, ubicación aproximada e información del dispositivo) para mejorar la experiencia de la aplicación.
  • Esta información se utiliza únicamente para la funcionalidad de la aplicación y análisis interno. No se comparte con terceros.

4. Propiedad intelectual

  • Todo el contenido y funcionalidad de la aplicación son propiedad de Quiet Mode.
  • No está permitido copiar, distribuir o modificar la aplicación sin autorización.

5. Limitación de responsabilidad

  • La Aplicación se proporciona "tal cual" y no garantizamos que esté libre de errores o interrupciones.
  • No somos responsables de daños directos o indirectos que surjan del uso de la aplicación.

6. Datos de usuario y privacidad

  • Tu privacidad es importante para nosotros. Consulta nuestra Política de Privacidad para obtener información detallada sobre la recopilación y el uso de datos.
  • Tienes derecho a solicitar la eliminación de todos tus datos de nuestros sistemas contactándonos en la dirección de correo electrónico proporcionada a continuación.

7. Cambios en los términos

  • Podemos actualizar estos términos ocasionalmente. El uso continuo de la aplicación implica la aceptación de los términos actualizados.
